# Broader Enterprise Model Market Dispositions

Status: P13.27 evidence, reviewed July 12, 2026

The decision rule is deliberately narrower than “has an AI API.” A direct
Starmap source is accepted only when the provider exposes a current,
machine-readable shared model catalog with stable invocation identity and
authoritative price or deployment semantics. Arbitrary customer deployments,
community model registries, and products outside the LLM/routing domain remain
separate customer-inventory, enrichment, or operational facilities.

Private image-training/upload models use a different API and must remain excluded. P13.37 owns native parsing and live/fault proof. |

The rejected surfaces are not “missing compatibility.” Replicate/fal selected
records remain ordinary external enrichment, while Nebius/Baseten/Scaleway
customer deployment facilities remain eligible for the separate private
inventory product. GitHub Models retirement is lifecycle evidence, not a
migration promise or alias to Microsoft Foundry.
